# ADU & HOME ADDITION - Wolf Carpenters

Esta pasta contém as páginas de conversão para serviços de ADU, Home Additions e Home Conversions da Wolf Carpenters.

## Estrutura do Projeto

```
ADU-HOMEADDITION/
├── index.html                 # Página principal com links para os 3 serviços
├── home-additions.html        # Página de Home Additions
├── adu.html                   # Página de ADU (Accessory Dwelling Units)
├── home-conversions.html      # Página de Home Conversions
├── 0 - Brand Logo/
│   └── brand-logo-wolfcarpenters.png
└── README.md
```

## Páginas

### 1. Index.html - Página Principal
- Overview dos 3 serviços
- Cards clicáveis para cada serviço
- Design responsivo com grid layout
- CTA principal para conversão

### 2. Home Additions (home-additions.html)
Serviços incluídos:
- Room Additions (quartos, banheiros, escritórios)
- Second Story Additions (segundo andar)
- Garage Additions (garagens)
- Sunrooms & Patio Enclosures (varandas fechadas)

### 3. ADU - Accessory Dwelling Units (adu.html)
Tipos de ADU:
- Detached ADU (estrutura separada)
- Attached ADU (conectada à casa)
- Garage Conversion ADU (conversão de garagem)

Usos principais:
- Renda de aluguel
- Espaço para família
- Home office
- Estúdio criativo

### 4. Home Conversions (home-conversions.html)
Tipos de conversão:
- Garage Conversions (garagem para espaço habitável)
- Attic Conversions (sótão)
- Basement Finishing (acabamento de porão)

Ideias populares:
- Home Office
- Personal Gym
- Entertainment Hub
- In-Law Suite
- Creative Studio
